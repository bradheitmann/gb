## ğŸ“š Dialogue Guidelines

When writing agent exchanges:

1. **Maintain distinct voices** - Each agent should be immediately recognizable
2. **Let tension build** - The best exchanges have escalating energy
3. **Include callbacks** - Reference earlier events (the penguin, the confession.txt)
4. **Mix the tones** - Scandalous, funny, and spicy can coexist
5. **Show productive chaos** - The dysfunction actually leads to better code
6. **Let the slang hit** - Some agents nail it, some don't, that's the point
